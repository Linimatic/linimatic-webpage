# Meta Title & Description Templates

Templates for crafting SEO-optimized metadata for each Linimatic page type. All templates include character counts and keyword placement guidance.

## Title Tag Rules

- **Length:** 50-60 characters (Google truncates at ~60)
- **Format:** `{Primary Keyword} | Linimatic A/S` or `{Page Topic} — {Context} | Linimatic`
- **Primary keyword** within first 50 characters
- **Brand name** at the end for recognition

## Meta Description Rules

- **Length:** 150-160 characters (Google truncates at ~160)
- **Include:** Primary keyword, value proposition, call-to-action
- **Avoid:** Duplicate descriptions across pages, keyword stuffing
- **Purpose:** Drive click-through, not rankings (descriptions don't directly affect ranking)
